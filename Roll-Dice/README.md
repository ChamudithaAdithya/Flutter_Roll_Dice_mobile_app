# 🎲 Flutter Roll Dice Mobile App

## 📌 Description

A simple and interactive **Flutter mobile application** that simulates rolling a dice. Each tap generates a random dice face, making it useful for games, learning Flutter basics, or quick decision-making apps.

This project is built using Flutter and Dart, focusing on UI design, state management, and random number generation.

---

## 🚀 Features

* 🎲 Random dice roll on button/tap
* 🔄 Smooth UI updates with state changes
* 📱 Responsive mobile-friendly design
* ⚡ Lightweight and fast performance
* 🧠 Simple logic using Dart’s random generator

## 🎯 How It Works

* The app uses Dart’s `Random()` class
* On button press, a number between 1–6 is generated
* The UI updates dynamically to show the corresponding dice image
